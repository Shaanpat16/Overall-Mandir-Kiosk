# Neue Montreal Font Files

The Edison Kiosk design system uses **Neue Montreal** by Pangram Pangram Foundry.

## Required files

Place the following font files in this directory:

- `NeueMontreal-Regular.woff2`
- `NeueMontreal-Bold.woff2`

## Fallback

Until these files are added, the kiosk will use `system-ui, -apple-system, 'Segoe UI', Roboto, sans-serif` as a fallback stack.

## Licensing

Neue Montreal is a commercial typeface. Obtain a valid license from [Pangram Pangram](https://pangrampangram.com/products/neue-montreal) before deploying.
