# Display Mode Photo

This folder contains the photo that will be displayed when the kiosk is in "Single Photo Mode".

## Required Photo:
- **Filename:** `display-photo.jpg` or `display-photo.png`
- **Recommended:** 
  - Resolution: 1920x1080 (Full HD) or higher
  - Format: JPG or PNG
  - High quality image
- **Content:** Any photo you want to display full screen

## How to Add:
1. Save your photo as `display-photo.jpg` or `display-photo.png`
2. Place it in this folder (`/public/images/display-mode/`)
3. Use the admin panel toggle to switch between regular kiosk and photo display mode

## Usage:
- In the admin dashboard, use the "Display Mode" toggle
- Switch between "Kiosk Mode" (regular interface) and "Photo Mode" (single photo)
- The photo will display full screen when in Photo Mode
